# Soliton stability and resolution

This page is for orbital stability, spectral stability, asymptotic stability, modulation analysis, multi-soliton dynamics, and soliton resolution.

## Scope

Core subjects include linearized operators, constrained variational methods, Grillakis-Shatah-Strauss theory, Evans functions, Krein signatures, modulation equations, multi-soliton interaction, radiation damping, dispersive decay, and asymptotic stability.

## Relation to IST and RHP

Some soliton-resolution results are proved through inverse scattering and Riemann-Hilbert asymptotics, while others rely on PDE, spectral, and modulation methods. This page should help distinguish the methods and prerequisites.

## Candidate seed resources

The first verification pass should include Dmitry Pelinovsky, Todd Kapitula, Mathew Johnson, and course or workshop pages on nonlinear waves, stability, and dispersive PDE.
