# Inverse scattering transform

This topic page is for direct and inverse scattering methods for integrable evolution equations.

## Scope

Core subjects include Lax pairs, Jost solutions, scattering coefficients, reflection coefficients, discrete spectrum, norming constants, Marchenko equations, trace formulae, conservation laws, and reconstruction of solutions from scattering data.

Important model equations include KdV, mKdV, defocusing and focusing NLS, DNLS, sine-Gordon, Toda-type models, and AKNS systems.

## What a good resource should provide

A strong IST resource should explain both the spectral problem and the time evolution of scattering data. It should also make clear which function spaces are assumed, which boundary conditions are used, and how solitons appear as discrete spectral data.

## Candidate seed resources

The first verification pass should include Peter Perry lecture materials, Peter D. Miller course materials, KIT inverse scattering courses, and high-quality course notes from active researchers in integrable systems and dispersive PDE.
