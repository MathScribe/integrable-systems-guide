# Nonlinear steepest descent

This page is for Deift-Zhou nonlinear steepest descent, d-bar steepest descent, and related asymptotic methods for Riemann-Hilbert problems.

## Scope

Core subjects include stationary phase points, signature tables, conjugation, lens opening, scalar RHPs, partial transmission coefficients, pole removal, soliton cones, model RHPs, parametrices, error RHPs, and asymptotic regions.

## Direct starting links

1. [Perry Lecture 2: Large-Time Asymptotics for the Defocussing NLS Equation](https://www.ms.uky.edu/~perry/Tsinghua-2025/Tsinghua-Lecture-2.pdf). Use this as the first link from IST to long-time asymptotics.
2. [Miller Math 651, Winter 2018](https://websites.umich.edu/~millerpd/CurrentCourses/651_Winter18.html). Use the lectures around long-time asymptotics, RHP theory, and dbar-steepest descent.
3. [Deift, *Riemann-Hilbert Problems*](https://arxiv.org/abs/1903.08304). Use for the general RHP steepest-descent framework.
4. [Olver--Trogdon, *Nonlinear steepest descent and the numerical solution of Riemann-Hilbert problems*](https://arxiv.org/abs/1205.5604). Use when you want to see how deformations translate into numerical RHP computation.
5. [Dieng--McLaughlin--Miller, *Dispersive Asymptotics for Linear and Integrable Equations by the dbar Steepest Descent Method*](https://arxiv.org/abs/1809.01222). Use as a specialized dbar-steepest-descent reference.

## Reading goal

The goal is to bridge the gap between knowing the inverse scattering transform and being able to read long-time asymptotic papers in NLS, DNLS, mKdV, KdV, and related equations.

## What a good resource should provide

A strong resource should make the deformation logic explicit: which transformations are made, why the jump matrices decay in specific regions, which model problems remain, and how the error estimate closes.

## Reading checkpoints

- Can you draw or describe the signature table for the phase?
- Do you know which contour deformation makes which exponential term decay?
- Can you identify the model problem and the error problem?
- Can you state the small-norm estimate needed to close the asymptotics?
- Can you distinguish the pure radiation region, soliton cone, and transition regions?
