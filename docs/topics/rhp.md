# Riemann-Hilbert problems

This page is for Riemann-Hilbert problem methods as used in integrable systems, orthogonal polynomials, asymptotic analysis, and related spectral problems.

## Scope

Core subjects include matrix RHPs, jump contours, normalization conditions, pole conditions, residue conditions, Cauchy projection operators, Beals-Coifman formulations, vanishing lemmas, small-norm RHP theory, and model problems.

## Relation to IST

In many integrable PDE problems, the inverse scattering step is reformulated as a Riemann-Hilbert problem. This translation is often the point at which long-time asymptotic analysis becomes technically tractable.

## What a good resource should provide

A strong RHP resource should state analytic assumptions precisely, explain normalization and jump data, and show how uniqueness and solvability are obtained. For nonlinear steepest descent, it should also explain contour deformation and error estimates.

## Candidate seed resources

The first verification pass should include Tom Trogdon, Peter D. Miller, Deift-Zhou style lecture notes, and workshop slides or notes that explicitly develop RHP methods for integrable PDE.
