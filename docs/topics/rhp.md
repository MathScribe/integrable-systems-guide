# Riemann-Hilbert problems

This page is for Riemann-Hilbert problem methods as used in integrable systems, orthogonal polynomials, asymptotic analysis, and related spectral problems.

## Scope

Core subjects include matrix RHPs, jump contours, normalization conditions, pole conditions, residue conditions, Cauchy projection operators, Beals-Coifman formulations, vanishing lemmas, small-norm RHP theory, and model problems.

## Direct starting links

1. [Peter D. Miller Math 651, Winter 2018](https://websites.umich.edu/~millerpd/CurrentCourses/651_Winter18.html). Use lectures 10, 18--24, and the later NLS/RHP applications as the main course-based entry.
2. [Percy Deift, *Riemann-Hilbert Problems*](https://arxiv.org/abs/1903.08304). Use this for a compact, authoritative overview of RHPs and nonlinear steepest descent.
3. [Tom Trogdon homepage](https://faculty.washington.edu/trogdon/). Use the RHP masterclass notebooks for concrete examples: erfc, Airy, defocusing NLS, oscillatory Cauchy operators, and filtering.
4. [Olver--Trogdon, *Nonlinear steepest descent and the numerical solution of Riemann-Hilbert problems*](https://arxiv.org/abs/1205.5604). Use this after basic RHP vocabulary is in place.
5. [Trogdon--Deconinck finite-genus KdV RHP article](https://linkinghub.elsevier.com/retrieve/pii/S0167278913000444). Use as a specialized bridge to finite-gap and numerical RHP.

## Relation to IST

In many integrable PDE problems, the inverse scattering step is reformulated as a Riemann-Hilbert problem. This translation is often the point at which long-time asymptotic analysis becomes technically tractable.

## What a good resource should provide

A strong RHP resource should state analytic assumptions precisely, explain normalization and jump data, and show how uniqueness and solvability are obtained. For nonlinear steepest descent, it should also explain contour deformation and error estimates.

## Reading checkpoints

- Can you write the jump condition, orientation, normalization, and residue conditions?
- Do you know what changes when poles are removed?
- Can you distinguish a model RHP from an error RHP?
- Can you explain the role of Cauchy projectors and Beals-Coifman operators?
- Can you state what a small-norm RHP estimate actually estimates?
