# Integrable Systems Research Guide

This guide is a curated map of resources for research training in integrable systems, inverse scattering transform, Riemann-Hilbert problem methods, nonlinear steepest descent, DNLS/NLS, nonlinear dispersive PDE, soliton asymptotics, and related spectral and geometric techniques.

The central unit of curation is not a link but an annotated resource record. A useful record should tell the reader what the resource is, what mathematical background it assumes, which methods or equations it covers, where it fits in a learning path, and how reliable or current the link is.

## Start here

- Use [Direct resource links](resources/direct-links.md) when you want a fast clickable list of the most important pages, PDFs, course pages, and notes.
- Use [Learning paths](learning-paths/index.md) when you want a reading order for a specific goal.
- Use [Core resources v0.1](resources/core-v0-1.md) for the first small set of curated resources.
- Use [Zotero seed map](resources/zotero-seed-map.md) to see what has been extracted from the Zotero dump but not yet fully promoted.
- Use [Group knowledge base](group/index.md) for public group notes, member notes, reading projects, and eventually public project outputs.

## Current stage

The project is now in an early `v0.1` curation stage. It has a working site structure, initial verified resources, a direct-link layer, learning paths, a Zotero-derived priority map, and a separate space for public group/member material.

## Main topic areas

- Inverse scattering transform and direct/inverse scattering theory.
- Riemann-Hilbert problem formulations in integrable PDE.
- Deift-Zhou nonlinear steepest descent and long-time asymptotics.
- NLS, DNLS, AKNS-type systems, and related spectral problems.
- Nonlinear dispersive PDE from the PDE and harmonic-analysis viewpoint.
- Soliton stability, modulation, Evans functions, and soliton resolution.
- Finite-gap methods, Riemann surfaces, theta functions, and algebro-geometric solutions.
- Numerical and software resources for RHP, IST, and reproducible nonlinear-wave experiments.

## How this repository is organized

Narrative reading guides live under `docs/topics/` and `docs/learning-paths/`. Structured records live under `data/`. Maintenance notes and link-review logs live under `maintenance/`. Public group/member material lives under `docs/group/`.

The website is generated from the Markdown files in `docs/`. The YAML files in `data/` are the source of truth for structured metadata and can later be used to generate tables or filtered views.
