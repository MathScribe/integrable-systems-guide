# About / 关于

This project is a small research-oriented guide for integrable systems and related methods.

这是一个面向可积系统研究者的轻量论文雷达，重点是近期出现的强创新、新方向和新方法。

## Site model / 页面职责

- 首页显示最近 30 天的最终选择，按论文实际研究事件日期分周组织；
- 论文条目默认显示概览，完整的“研究问题与主要结果 / 可积结构与方法 / 创新”按需展开；
- 归档首页提供周级入口，周页面复用首页的同一条目组件；
- `data/papers.yml` 保留历史书目信息，`data/radar_papers.yml` 保存当前近期雷达需要的书目覆盖；
- `data/editions.yml` 保存研究事件日期、筛选说明和当次注释；
- 页面由数据文件统一生成，不手工分别维护首页和归档。

每日运行是维护频率，不要求每日发布。日期表示论文首次公开、重大修订或正式期刊在线发表的实际事件日期，不是本站推荐日期。

## Curation policy / 策展原则

本站面向可积系统读者。主题相关性采用宽松边界，创新强度是主要判断。引力、几何、概率、随机矩阵、量子多体、统计物理、光学、实验和数据驱动工作，只要可积结构参与核心结果，并且能够清楚说明值得关注的新方向或方法，都可以收录。

内部审核主要回答两个问题：

1. 论文是否提出、揭示或实质扩展了可积结构或方法？
2. 即使结构已有，它是否不可替代地产生了清楚、可核验的创新？

这两个问题不生成公开等级标签，也不要求把每篇论文硬分为唯一类别。通常不收录机械套用标准变换、只有少量低阶显式解或参数图、可积性仅作为背景、以及无法可靠说明结构作用和创新的内容。

## Annotation format / 注释格式

每篇论文默认显示：

1. 实际研究事件日期；
2. 一到两个官方 arXiv 分类；
3. 最多两个受控结构或主题标签；
4. 作者、标题和简明论文概览；
5. arXiv、PDF、DOI 或期刊页面链接。

展开后统一说明：

- **研究问题与主要结果**
- **可积结构与方法**
- **创新**

标签来自受控词表，不临时创造含义不稳定的缩写。

## Metadata and history / 元数据与历史

书目事实使用 arXiv ID、DOI 和规范化题名去重。旧预印本首次正式在线发表时可以作为新的研究事件进入首页，同时注明原始 arXiv 首次公开日期。普通语言润色、参考文献更新和元数据修正不作为重大修订重复收录。

周页面只公开最终选择和聚合筛选说明，不公开逐篇未收录名单，也不对未收录论文作负面评价。Git 历史保留编辑变更记录。

## AI-assisted curation / AI 辅助整理

AI 用于候选发现、去重、元数据清洗、初步筛选和文字整理。论文说明会核对官方 arXiv、论文正文、DOI 或出版商页面；除特别标注外，不代表编辑已经完整阅读或逐句审校，也不构成对数学正确性的独立验证。数学结论请以原论文为准。
