# About / 关于

This project is a small research-oriented guide for integrable systems and related methods.

这是一个面向可积系统研究者的轻量论文雷达，重点是近期值得关注的新方向、新结构和新方法，而不是建设完整论文数据库或 arXiv 镜像。

## Site model / 页面职责

- 首页展示最近 30 天的精选论文，按周组织；
- 每篇论文默认只显示书目信息、受控标签和一段论文概览；
- “研究问题与主要结果 / 可积结构与方法 / 创新”按需展开；
- Radar 归档首页按周提供入口，周页面保存该周最终选择和聚合筛选说明；
- `data/papers.yml` 保存当前书目信息，`data/editions.yml` 保存研究事件、说明和推荐历史；
- 页面由 renderer 统一生成，不手工分别维护首页和周归档。

每日运行是维护频率，不要求每天发布。公开排序使用论文的实际研究事件日期：首次公开、重大修订或首次正式在线期刊发表；本站收录日期只属于编辑历史。

旧的逐日 edition 记录继续保留为编辑历史，但新的公开首页和周归档只由 `frontier` 数据生成。结构标签只能从受控词表中选择，不能用标题片段或临时造词代替标准术语。

## Curation policy / 策展原则

本站面向已有可积 PDE、谱方法或相关数学物理背景的读者。主题相关性是宽松入口，创新强度是主要判断。

候选论文应当能够说明：

1. 研究问题与主要结果是什么；
2. 可积结构或方法在结果中起什么实际作用；
3. 创新是否清楚、可核验，并足以值得进一步阅读。

不要求每篇论文都提出新的 Lax pair。已有 Darboux、IST、RHP、有限带、Yang--Baxter 或精确解结构，只要产生系统的新分类、任意阶解族、新渐近、新拓扑或物理机制、新反演、控制、实验或统计方法，也可以收录。

引力、几何、概率、随机矩阵、量子多体、统计物理和光学等方向，不因距离经典可积 PDE 较远而自动排除。可积性仅作为背景、标准方法的机械套用、少量低阶解图或无法可靠解释创新的工作通常不公开推荐。

## Annotation format / 注释格式

论文条目采用稳定的三部分说明：

- **研究问题与主要结果**：论文解决什么问题，得到什么主要结论；
- **可积结构与方法**：Lax、散射、RHP、层级、Hamiltonian、Darboux、有限带等结构如何参与结果；
- **创新**：相对于已有工作的新增部分，不夸大长期影响。

默认概览用于快速筛选，完整说明用于深入了解。公开页面不使用“核心/相邻前沿”或其他容易被理解为质量等级的分类。

## Metadata and history / 元数据与推荐历史

`data/papers.yml` 是当前书目事实的注册表，用 arXiv ID、DOI 和规范化题名去重。`data/editions.yml` 保存研究事件日期、说明和推荐历史。CI 检查数据引用、去重、标签、周编号和生成结果一致性。

旧预印本首次正式在线发表时可以作为新的研究事件进入首页，同时显示期刊发表日期和原始 arXiv 首次公开日期。普通语言润色、元数据修正和勘误不视为重大修订。

## AI-assisted curation / AI 辅助整理

AI 用于候选发现、去重、元数据清洗、初步筛选和文字整理。论文说明经过元数据核验；除特别标注外，不代表编辑已经完整阅读或逐句审校。数学结论和优先权判断请以原论文为准。

## Design references / 设计参考

本站借鉴成熟论文列表的高信息密度和快速浏览方式，但不复制其数据库、全文搜索、作者网络或 feed 系统。Leonid Petrov 的网站等专题资源只用于人工查漏和资料导航，不被镜像或自动同步。

本站不是课题组官方网站，也不代表任何作者、研究组或机构的正式立场。
