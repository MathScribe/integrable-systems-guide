## Literature search / 文献检索

- [arXiv Search](https://arxiv.org/search/)：检索预印本，可按题名、作者、摘要和分类筛选。
- [MathSciNet Publications Search](https://mathscinet.ams.org/mathscinet/publications-search)：检索数学论文、正式出版信息和评论。
- [zbMATH Open](https://zbmath.org/)：检索数学文献、作者、分类和相关条目。
- [Google Scholar](https://scholar.google.com/)：适合广泛检索、引用追踪和查找不同版本。
- [Semantic Scholar](https://www.semanticscholar.org/)：适合查看相关论文、参考文献和引用关系。
- [Crossref Metadata Search](https://search.crossref.org/)：用于核对 DOI、期刊、出版日期和正式书目信息。
