# Software

This page will collect a small number of software resources useful for research training in integrable systems, inverse scattering, Riemann-Hilbert problems, nonlinear waves, and reproducible numerical experiments.

## Inclusion criteria

A software resource should be included only if its research use is clear. Examples include numerical RHP solvers, spectral collocation packages, Fourier pseudospectral solvers, Chebfun-based experiments, reproducible code accompanying papers, and computational notebooks for soliton dynamics.

## Required annotation

Each software entry should explain:

- what mathematical problem it helps investigate;
- what equations or methods it supports;
- whether it is actively maintained;
- what dependencies it requires;
- whether it has examples or reproducibility scripts;
- what its limitations are.

Do not include generic programming tutorials unless they are directly tied to research workflows in nonlinear waves or integrable systems.
