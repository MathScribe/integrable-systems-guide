# Group knowledge base

This section is reserved for material produced by the group or by individual members, provided the material is public and the author wants it linked here.

The main guide should remain a curated map of external research-training resources. Group material should be kept in a separate section so that readers can distinguish between established external references and local notes, seminar writeups, reading logs, code, and project pages.

## What belongs here

Suitable items include:

- public course notes;
- seminar writeups;
- reading-group notes;
- reproducible computation notebooks;
- public project pages;
- public bibliographies or Zotero-derived reading maps;
- slides from group talks, when the speaker agrees.

Do not link private drafts, unpublished student work, internal meeting notes, or anything whose author has not agreed to public listing.

## Proposed structure

- Public work: public papers, preprints, seminar projects, and reproducibility repositories.
- Member notes: public personal note repositories and lecture-note pages.
- Reading projects: group reading logs organized by topic.

The metadata source for this section should eventually live under `data/group-public-work.yml` and `data/member-notes.yml`.
