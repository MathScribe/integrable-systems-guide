# Reading projects

This page is for group reading projects and seminar reading logs.

A reading project should be small enough to be useful. Prefer a focused topic such as "defocusing NLS long-time asymptotics" or "finite-gap Riemann surfaces for mKdV" rather than a broad area such as "integrable systems".

## Suggested format

```yaml
- id:
  title:
  organizer:
  participants:
  dates:
  topic:
  target_papers:
  prerequisite_resources:
  meeting_notes:
  public_outputs:
  status:
```

## Candidate projects

- PDE background to IST/RHP.
- DNLS inverse scattering and long-time asymptotics.
- Riemann-Hilbert problem basics and nonlinear steepest descent.
- Finite-gap solutions, Riemann surfaces, and theta functions.
- Soliton stability and Evans-function methods.

Keep project notes public only when all participants agree.
