# Public work

This page will list public work by the group or its members when it is directly relevant to the guide.

The goal is discoverability, not self-promotion. Each entry should explain how the work connects to integrable systems, IST/RHP, nonlinear dispersive PDE, DNLS/NLS, soliton stability, finite-gap methods, or reproducible computation.

## Entry types

- Preprints and papers.
- Seminar reports.
- Lecture notes.
- Reproducibility code.
- Numerical experiments.
- Reading-project outputs.
- Slides from public talks.
