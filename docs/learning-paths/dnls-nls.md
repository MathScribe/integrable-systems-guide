# DNLS/NLS research preparation

This path is for preparing to read papers on NLS, DNLS, inverse scattering, Riemann-Hilbert problems, global well-posedness, and long-time asymptotics.

## 1. PDE-side preparation

Read enough nonlinear dispersive PDE to understand the function-space assumptions in the scattering theory.

Useful resources:

- Monica Visan's expository notes for critical NLS and dispersive PDE.
- Tao's Sobolev, Fourier analysis, and PDE blog/course notes.
- Linares--Ponce as a textbook reference for nonlinear dispersive equations.
- Michael Taylor notes for PDE and functional-analysis background.

## 2. IST-side preparation

Begin with NLS rather than DNLS.

Useful resources:

- Perry's inverse-scattering lectures and expository notes.
- Miller Math 651 lectures on defocusing and focusing NLS.
- Deift's RHP notes for the RHP language.

Key checkpoints:

- direct scattering and Jost solutions;
- reflection coefficient and discrete spectrum;
- time evolution of scattering data;
- RHP reconstruction;
- pole removal and pure soliton RHPs;
- long-time asymptotic regions.

## 3. DNLS-specific issues

DNLS should be treated after NLS because the spectral problem and gauge structure introduce extra technical layers.

Track these themes separately:

- Kaup-Newell spectral problem;
- gauge equivalence among DNLS-type equations;
- weighted Sobolev scattering maps;
- solitons and threshold phenomena;
- RHP normalization and singularities near the origin;
- long-time asymptotics in soliton and radiation regions.

## 4. Bibliography signals

The Zotero dump contains a substantial DNLS cluster, including DNLS well-posedness, inverse scattering, long-time asymptotics, and Pelinovsky-related references. These should be used to infer paper clusters, but they should not all be promoted to resource pages.

Promote only papers that play one of these roles:

- foundational method paper;
- unusually clear exposition;
- direct prerequisite for a current project;
- canonical reference repeatedly used in later work.
