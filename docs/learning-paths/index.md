# Learning paths

This section turns the resource guide into research-training routes. A path is not a syllabus in the formal sense; it is a curated order of resources for a specific mathematical goal.

The current paths are provisional. They should be revised as the resource metadata improves.

## Available paths

- From PDE background to IST/RHP.
- From Riemann-Hilbert problems to nonlinear steepest descent.
- DNLS/NLS research preparation.
- Finite-gap and algebro-geometric methods.

## How to read a path

Each path distinguishes between:

- prerequisite repair: background needed before the main topic;
- core training: resources to read carefully;
- paper preparation: resources that prepare the reader for current research papers;
- reference shelf: books, surveys, and papers to consult as needed.

Do not try to read every resource linearly. Use the path to decide what to read next for a specific problem.
