# Core topics / 核心主题

This page records the current scope of the guide. It is not meant to be a textbook or encyclopedia.

本页只列出当前指南覆盖的研究主题，不展开成百科式说明。

- Inverse scattering transform / 反散射变换
- Riemann-Hilbert problems / 黎曼-希尔伯特问题
- Nonlinear steepest descent / 非线性最速下降法
- NLS, DNLS, KdV, mKdV, and AKNS-type systems
- Rogue waves, breathers, solitons, and nonlinear waves / 怪波、呼吸子、孤子与非线性波
- Finite-gap and algebro-geometric methods / 有限带与代数几何方法
- Spectral and orbital stability / 谱稳定性与轨道稳定性
- Numerical and data-driven methods for nonlinear PDE / 非线性偏微分方程的数值与数据驱动方法

A separate topic page should be created only when there is enough curated material to justify it.
