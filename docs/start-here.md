# Start here

This guide is designed for research training. The best way to use it is to choose a mathematical task first, then follow the shortest route that prepares you for that task.

## Typical use cases

A reader might use the guide to:

- prepare for a paper using inverse scattering or Riemann-Hilbert methods;
- repair background in spectral theory, Fourier analysis, complex analysis, or asymptotic analysis;
- compare PDE-based and IST-based approaches to NLS, DNLS, KdV, mKdv, or related equations;
- find a course page, lecture-note set, summer school, or monograph before starting a reading project;
- organize public group notes, seminar writeups, and reproducible computations around a shared topic.

## Resource levels

Curated resources should eventually be assigned one of these levels.

| Level | Meaning |
|---|---|
| `introductory` | Useful for first contact with a topic. |
| `graduate-course` | Suitable for systematic study with standard graduate prerequisites. |
| `paper-preparation` | Useful immediately before reading research papers. |
| `research-reference` | Useful as a technical reference during research. |
| `specialized` | Valuable but focused on a narrow method, equation, or literature branch. |

## Quality grades

The intended grade scale is:

| Grade | Meaning |
|---|---|
| `S` | Core training resource; recommend prominently and maintain carefully. |
| `A` | High-quality resource for a major topic or method. |
| `B` | Useful local resource for a chapter, lecture, method, or example. |
| `C` | Background or auxiliary resource. |
| `D` | Candidate resource that still needs review. |
| `reject` | Reviewed and not suitable for inclusion. |

## Contribution rule

Do not add bare links unless they are explicitly marked as unreviewed candidates. A mature entry should explain why the resource is useful, what mathematical background it assumes, and where it fits in the guide.

For now, keep the core small. It is better to have ten annotated, reliable resources than one hundred unreviewed links.
