# Dead links and moved resources

Use this file to track resources whose original links have changed or disappeared.

Suggested statuses:

- `alive`: current URL works.
- `redirected`: current URL redirects but still reaches the resource.
- `moved`: resource was found at a new canonical URL.
- `partial`: landing page exists but some PDFs, slides, or videos are missing.
- `dead`: link is unavailable and no replacement has been found.
- `archived-only`: resource is only accessible through an archive.
- `restricted`: resource requires login or institutional access.
